This is a short python script that helps create basic java classes which are usually linked together: Bean - EJB, EJBIF - Helper, and if needed it can also insert the new EJB into the jboss-ejb3.xml and weblogic-ejb-jar.xml.
You can define your class' name and package it should be put in and also choose which modules you wish to create.

Install Python 3.6.3+ (For easier usage add to the PATH)

#We have to install pyqt5
Type in cmd 'pip3 install pyqt5'

Run templater.py

Edit config file for your own needs.
